# Recipe UA — веб‑додаток для підбору рецептів за інгредієнтами + калорійність

Це **простий** навчальний проєкт (курсова): інтерфейс українською, пошук рецептів за інгредієнтами, перегляд деталей страви, підрахунок **орієнтовних** калорій за інгредієнтами.

## Які API використані (безкоштовні)
1) **TheMealDB** — рецепти (безкоштовно, без ключа для навчальних проєктів)  
Документація: https://www.themealdb.com/api.php

2) **USDA FoodData Central (FDC)** — калорійність продуктів (безкоштовний API‑key)  
Сторінка ключа: https://fdc.nal.usda.gov/api-key-signup  
API guide: https://fdc.nal.usda.gov/api-guide

> Якщо ключа FDC немає — додаток покаже підказку і порахує калорії **приблизно** на базі вбудованого міні‑словника.

## Запуск
```bash
npm i
cp .env.example .env
# встав VITE_FDC_API_KEY=... (опційно, але бажано)
npm run dev
```

## Як працює пошук
- Вводиш інгредієнти через кому: `курка, рис, помідор`
- Додаток робить запити до TheMealDB `filter.php?i=...` для кожного інгредієнта і знаходить **спільні** страви.
- По кліку відкривається детальна сторінка з інструкціями та інгредієнтами.
- Калорії: для кожного інгредієнта робиться пошук у FDC і береться `Energy (kcal)`; далі сумується.

## Структура
- `src/services/mealdb.js` — запити до TheMealDB
- `src/services/fdc.js` — запити до USDA FDC (калорії)
- `src/pages/*` — сторінки
- `src/assets/*` — картинки/логотип
